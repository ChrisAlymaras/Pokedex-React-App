Pokedex app is a web application that fetches live data from the PokeAPI and presents it in an interactive, user-friendly interface.

**Highlights** :

- **Interactive Pokemon cards** with header, id, type badges, and official artwork images.
- **Expandable view** with tabs for:
  - Info (height, weight, abilities)
  - Stats (HP, Attack, Defense, etc.)
  - Evolution chain (visual progression images).
- **Responsive design** for desktop and mobile.
- **Dynamic colors** based on Pokemon type (grass,water etc.).
- **Pokeball background** for extra flair.
